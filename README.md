# 🚀 MediCare – Complete Healthcare & Patient Management Platform

A scalable healthcare ecosystem built with modern technologies to manage patients, hospitals, clinics, appointments, rewards, communication, and real-time healthcare operations.

This platform includes:
- Multi-role admin panels
- Mobile APIs
- Real-time notifications
- Video consultations
- Appointment management
- Reward & loyalty systems
- Payment gateway integrations
- Cloud storage
- Cron automation
- CMS management
- Activity monitoring
- Enterprise security

php artisan migrate

php artisan serve
